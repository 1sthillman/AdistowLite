# AdistowLite - Müşteri Paneli

RestoQR projesinin müşteri tarafı arayüzü.

## Teknolojiler
- Next.js 14
- Tailwind CSS
- Firebase
- Framer Motion

## Özellikler
- QR Kod ile menü görüntüleme
- Masa bazlı sipariş ve garson çağırma
- Değerlendirme sistemi
- Çoklu dil desteği

## Kurulum
1. `npm install`
2. `.env.local` dosyasını oluşturun ve Firebase anahtarlarınızı ekleyin.
3. `npm run dev`
